About Me

Welcome to my portfolio page. I would like to first thank you for taking the time to learn more about who I am. I love to build and scale companies. I made a list of important qualities I wanted from a full-stack coding program. After doing extensive research, I found that UC San Diego Extention, Full-Stack Coding Bootcamp program was the program that gave me the best opportunity to achieve my goals. So here I am, uploading code I am writing for class projects and homework assignments.
